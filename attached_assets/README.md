# Attached Assets

This folder contains visual assets used to explain the Artemis ecosystem and MVP progress.

## What you will find here
- Architecture diagrams (Hunt ↔ ARISE ↔ Gate)
- Screenshots of key flows (proposal creation, stage-based review, dashboards)
- Supporting visuals for product documentation and recruiting review

## Naming convention
- `diagram_ecosystem_arise.png`
- `screenshot_gate_overview.png`
- `screenshot_hunt_proposal_capture.png`

Keep filenames descriptive and stable so they can be referenced from README and docs.
